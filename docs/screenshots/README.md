# レビュー用スナップショット (SOT-1609)

Window Runner の各画面をヘッドレスブラウザ (Playwright / Chromium) で実描画してキャプチャした
レビュー用スクリーンショット。Canvas を native 解像度 480×270 で描画し、720×405 表示・
device scale 2 で保存している。

各プレイ画面は **乗り物の中から外を覗いている車内ビュー** になっている。厚いキャビン枠が
プレイ画面を囲み、下部にはステージごとの車内インテリア(車=ダッシュボード/ハンドル・
計器、宇宙=操作パネル・スクリーン、その他=前席のヘッドレスト)が並ぶ。窓の内側には
ガラスの映り込みとビネット(周辺減光)が入り、囲われた車内から窓の外を見ている奥行きを
表現している。飛行機・宇宙は窓の角が大きく丸い(丸窓/舷窓)。

| ファイル | 画面 |
| --- | --- |
| `00_home_stage_select.png` | ホーム(ステージ選択) — 進捗・ロック/挑戦可能/クリア済み表示 |
| `01_stage1_walk.png` | ステージ1「徒歩」— 街並み背景 / 車内から窓の外(前席ヘッドレスト) |
| `02_stage2_car.png` | ステージ2「車」— ガードレール / 運転席から窓の外(ダッシュボード・ハンドル) |
| `03_stage3_train.png` | ステージ3「電車」— 架線・線路 / 車内から窓の外(前席ヘッドレスト) |
| `04_stage4_shinkansen.png` | ステージ4「新幹線」— 防音壁 / 車内から窓の外(前席ヘッドレスト) |
| `05_stage5_airplane.png` | ステージ5「飛行機」— 滑走路 / 客席の丸みのある窓から外 |
| `06_stage6_space.png` | ステージ6「宇宙」— 星空 + 惑星 / 宇宙船の舷窓・操作パネルから外 |

各プレイ画面はステージ開始イントロ(84フレーム)直後、開始直後の安全地帯(0–420px)上で
1フレームを描画したもの。
